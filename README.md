# IFCS---Animal-Quiz-
As part of the formative assessment, this repo contains a small quiz demonstrating my understanding of programming language. Making use of functions, loops and modules. 

# User Documentation 

In order to run this you will need Python downloaded on your computer. You can then run the quiz directly in Python. Here is how:

## Step 1: Download the file found on this REPO titled FA Animal Quiz.
Save the file within your local harddrive to be able to access the quiz.

## Step 2: Find the file in your local harddrive.
Once you have saved the file, you can now open it in order to run it. Do this by right clicking on the file within the folder and select "open with".

## Step 3: Select open with python. 
After selecting "open with" select "Python". 
The quiz will automatically open.

## Step 4: Follow the instructions on the screen.
The quiz will now guide you through. You will start off by entering your name, then selecting the answers you believe are correct.
You you should see a cursor on your screen which is where you can input your answer. 
If you wish to redo the quiz, you will be able to. 


# Technincal Documentation 
This is a simple quiz built to demonstrate my understanding and ability to use functions, loops and one module. It is designed to be intuitive and to minimise errors. Error handling has been included to streamline the experience. For this reason there a lots of dividers to increase readability. 
It is short 4 question quiz that can be played directly in the Python terminal. 

## Features: 
* Robust input validation,
* Randomised responses to user answers,
* Loop to allow user to try again,
* Tracked answers to provide an overall score at the end,
* Clean output for user experience and correct answers provided if user answer is incorrect,
* No additional packages/extensions or additional applications needed to run the quiz. However, Visual Studio Code can also be used to run and edit the code.

## Requirements:
* **Python:** version 3.8 or later
* **Operating System (OS):** Windows, macOS or Linux
* No additional packages/extensions or additional applications needed to run the quiz. However, Visual Studio Code can also be used to run and edit the code.
